# 💬 Chat Application

A simple real-time Chat Application built using modern web technologies. It allows users to send and receive messages through an interactive and responsive interface.

## 🚀 Features

- 💬 Real-time messaging
- 👤 User-friendly interface
- 📱 Responsive design
- ⚡ Fast and smooth performance

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- Socket.IO
- MongoDB
- CSS / Tailwind CSS

## 📂 Project Structure

```
chatApplication/
│── client/
│── server/
│── package.json
│── README.md
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/InsaanAyu/chatApplication.git
```

2. Navigate to the project

```bash
cd chatApplication
```

3. Install dependencies

```bash
npm install
```

4. Start the application

```bash
npm start
```
